# Procédure de gestion des incidents de paie et rectification

## 1. Objet

Cette procédure décrit les actions à suivre en cas de problème lié à la paie, notamment :

- salaire non reçu  
- erreur sur le bulletin de paie  
- écart de rémunération  

Elle précise les étapes de signalement, de traitement et de correction.

---

## 2. Types d’incidents de paie

Les incidents les plus fréquents sont :

### 2.1 Paie non reçue
- salaire non crédité à la date prévue  
- virement rejeté ou retardé  

### 2.2 Erreur sur le bulletin de paie
- montant incorrect  
- absence ou erreur de prime  
- erreur sur heures supplémentaires  
- retenue injustifiée  

### 2.3 Variation inexpliquée
- écart significatif par rapport au mois précédent  
- incohérence avec les éléments connus (présence, primes, etc.)

---

## 3. Gestes immédiats à effectuer

Avant de contacter le service RH, l’employé doit :

1. Vérifier son bulletin de paie  
2. Vérifier son compte bancaire (date de valeur, délai bancaire)  
3. Vérifier ses coordonnées bancaires enregistrées  
4. Identifier les éléments variables (primes, absences, heures)  

Ces vérifications permettent d’éviter les faux incidents.

---

## 4. Ouverture d’un ticket paie

Si le problème persiste, l’employé doit ouvrir une demande via :

- le portail RH (SIRH), rubrique *Support / Paie*  
ou  
- le système de ticketing interne  

La demande doit contenir :

- identité de l’employé  
- mois concerné  
- description précise du problème  

---

## 5. Justificatifs à fournir

Selon le type d’incident, les documents suivants peuvent être demandés :

- bulletin de paie concerné  
- relevé bancaire (en cas de non réception)  
- justificatif d’heures supplémentaires  
- validation de primes ou objectifs  
- justificatif d’absence ou de présence  

Ces éléments permettent d’accélérer le traitement.

---

## 6. Traitement par le service RH

Le service RH effectue les actions suivantes :

1. Analyse du bulletin et des données de paie  
2. Vérification auprès du manager si nécessaire  
3. Identification de l’origine de l’erreur  
4. Détermination de la correction à apporter  

Le manager peut être sollicité pour valider certains éléments (heures, performance, etc.).

---

## 7. Délai de régularisation

Les délais de traitement sont généralement :

| Type d’incident | Délai moyen |
|----------------|------------|
| Vérification simple | 2 à 3 jours ouvrés |
| Correction avec validation | 3 à 7 jours ouvrés |
| Cas complexe | jusqu’à 10 jours ouvrés |

Les délais peuvent varier selon la complexité du dossier.

---

## 8. Modalités de correction

En cas d’erreur confirmée, la correction peut prendre plusieurs formes :

- régularisation sur le bulletin du mois suivant  
- versement exceptionnel (virement complémentaire)  
- ajustement administratif  

---

## 9. Émission d’un nouveau bulletin

Si nécessaire :

- un **bulletin de paie corrigé** peut être émis  
- il remplace ou complète le bulletin initial  
- il est mis à disposition dans le SIRH  

---

## 10. Cas d’escalade

Une escalade peut être nécessaire si :

- le problème n’est pas résolu dans les délais  
- l’impact financier est important  
- le cas nécessite une validation exceptionnelle  

Dans ce cas :

- le dossier est transmis à un niveau RH supérieur  
- une priorité de traitement est appliquée  

---

## 11. Suivi de la demande

L’employé peut :

- suivre l’état de sa demande via le système de ticketing  
- recevoir des mises à jour du service RH  
- être contacté pour complément d’information  

---

## 12. Résumé opérationnel

- Vérifier d’abord les éléments de base  
- Ouvrir un ticket si le problème persiste  
- Fournir les justificatifs nécessaires  
- Le service RH analyse et corrige si besoin  
- Une régularisation ou un nouveau bulletin peut être émis
