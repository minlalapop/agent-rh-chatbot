# Calendrier des jours fériés et absences collectives

## 1. Objet

Ce document présente :

- les jours fériés officiels  
- les éventuels jours chômés au sein de l’entreprise  
- les périodes de fermeture collective  
- les ponts éventuels  

---

## 2. Jours fériés officiels

Les jours fériés sont définis selon la réglementation en vigueur dans le pays.

Exemples de jours fériés (liste indicative) :

- Nouvel An  
- Fête du travail  
- Fête nationale  
- Fêtes religieuses (selon calendrier officiel)  

Les jours fériés peuvent être :

- chômés (non travaillés)  
- ou travaillés selon les besoins de l’activité  

---

## 3. Jours chômés dans l’entreprise

Certains jours fériés peuvent être :

- systématiquement chômés  
- ou aménagés selon les politiques internes  

Ces jours sont communiqués en début d’année ou via le SIRH.

---

## 4. Fermetures collectives

L’entreprise peut prévoir des périodes de fermeture collective, par exemple :

- périodes estivales  
- fin d’année  

Pendant ces périodes :

- les employés doivent poser des congés  
- ou utiliser des jours disponibles (congés, RTT)

Les dates sont communiquées à l’avance.

---

## 5. Ponts

Un pont correspond à un jour non travaillé situé entre :

- un jour férié  
- et un week-end  

Exemples :

- vendredi après un jeudi férié  
- lundi avant un mardi férié  

Les ponts peuvent être :

- accordés par l’entreprise  
- ou posés individuellement en congé  

---

## 6. Spécificités locales

Les jours fériés et leur organisation peuvent varier selon :

- le pays  
- la région  
- le site de travail  

Certaines entités peuvent bénéficier de calendriers spécifiques.

---

## 7. Communication et visibilité

Les informations sont disponibles via :

- le SIRH (calendrier RH)  
- les communications internes  
- le manager  

Les employés sont invités à consulter régulièrement ces informations.

---

## 8. Impact sur les congés

- Les jours fériés chômés ne sont pas déduits des congés  
- Les fermetures collectives peuvent nécessiter l’utilisation de jours de congé  
- Les ponts peuvent impacter le solde de congés ou de RTT  

---

## 9. Résumé opérationnel

- Les jours fériés sont définis par la réglementation  
- Certains jours peuvent être chômés dans l’entreprise  
- Des fermetures collectives peuvent être imposées  
- Les ponts peuvent être accordés ou posés  
- Les informations sont disponibles dans le SIRH
