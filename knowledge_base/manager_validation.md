# Validation manager et organisation

## Circuit d'approbation des congés
Le manager valide les demandes de congés dans le SIRH selon la charge de l'équipe, les priorités opérationnelles et les règles d'équité entre salariés.

## Organisation hiérarchique
Le manager direct et le responsable N+1 sont maintenus dans l'annuaire interne. En cas de changement d'équipe, la mise à jour relève du référentiel RH.
