# Procédure d’Accès aux Documents Contractuels

## 1. Objet

Ce document décrit les modalités d’accès aux documents contractuels et administratifs des salariés, ainsi que les procédures de récupération en cas d’indisponibilité.

---

## 2. Accès au contrat de travail

Le contrat de travail est accessible via :

- le **portail salarié (SIRH)**
- le **coffre-fort numérique**

### Étapes :

1. Se connecter au portail RH (SIRH)
2. Accéder à la section "Mes documents"
3. Sélectionner "Contrat de travail"
4. Télécharger le document

---

## 3. Accès à l’espace salarié (SIRH)

Le SIRH est l’outil principal pour consulter les documents RH.

Il permet :

- l’accès aux contrats
- la consultation des bulletins de paie
- le suivi des demandes RH
- la mise à jour des données personnelles

L’accès se fait via :

- identifiants professionnels
- authentification sécurisée

En cas de problème d’accès, contacter le support interne.

---

## 4. Coffre-fort numérique

Le coffre-fort numérique permet le stockage sécurisé des documents :

- contrats de travail
- avenants
- attestations
- documents administratifs

Caractéristiques :

- accès personnel et sécurisé
- archivage à long terme
- disponibilité même après départ de l’entreprise (selon conditions)

---

## 5. Récupération d’une attestation employeur

Les attestations peuvent être obtenues via :

- le SIRH (si disponible)
- une demande auprès du service RH

### Procédure :

1. Vérifier la disponibilité dans le SIRH
2. Si indisponible, faire une demande via :
   - portail RH (ticket)
   - email RH
3. Préciser :
   - type d’attestation (emploi, salaire, etc.)
   - motif de la demande

---

## 6. Formats de documents

Les documents RH sont généralement disponibles en :

- format PDF (principal)
- format téléchargeable et imprimable

Les documents officiels sont signés et horodatés si nécessaire.

---

## 7. Délais de génération

| Type de document | Délai moyen |
|----------------|------------|
| Document disponible (SIRH) | immédiat |
| Attestation standard | 2 à 5 jours ouvrés |
| Document spécifique | 3 à 7 jours ouvrés |

Les délais peuvent varier selon la nature de la demande.

---

## 8. Document indisponible

Si un document n’est pas accessible :

- vérifier les filtres ou l’interface SIRH
- s’assurer que le document a bien été généré
- contacter le service RH

---

## 9. Contact RH

En cas de problème :

- Email : rh@entreprise.com
- Portail RH (ticket)
- Support interne

La demande doit inclure :

- nom et identifiant salarié
- document demandé
- urgence éventuelle

---

## 10. Résumé opérationnel

- Les documents sont accessibles via le SIRH ou le coffre-fort numérique
- Le contrat de travail est disponible dans "Mes documents"
- Les attestations peuvent être téléchargées ou demandées
- Les délais varient selon le type de document
- Le service RH peut être contacté en cas de problème
