# Procédure de demande de congé

## 1. Objet

Cette procédure décrit les étapes à suivre pour effectuer une demande de congé, depuis la saisie jusqu’à la validation.

---

## 2. Comment poser un congé

Pour poser un congé, l’employé doit :

1. Se connecter au portail RH (SIRH)  
2. Accéder à la rubrique *Congés et absences*  
3. Sélectionner les dates souhaitées  
4. Vérifier son solde disponible  
5. Soumettre la demande  

Une fois la demande envoyée, elle est en attente de validation.

---

## 3. Outil utilisé

Les demandes de congé sont gérées via :

- le **SIRH (portail salarié)**  
- éventuellement une application mobile RH  

Toute demande doit être saisie dans cet outil pour être prise en compte.

---

## 4. Validation par le manager

- Le manager (N+1) est responsable de la validation  
- Il vérifie :
  - la disponibilité de l’employé  
  - les contraintes de l’équipe  
  - la cohérence avec les périodes d’activité  

La demande peut être :

- validée  
- refusée  
- ou modifiée  

---

## 5. Délai de prévenance

Les congés doivent être demandés avec un délai raisonnable.

Recommandations générales :

| Type de congé | Délai recommandé |
|---------------|----------------|
| Congé court (1–2 jours) | quelques jours à l’avance |
| Congé long | plusieurs semaines à l’avance |

Le délai peut varier selon les contraintes du service.

---

## 6. Cas urgents

En cas de situation exceptionnelle :

- l’employé peut informer directement son manager  
- la demande doit être régularisée dans le SIRH dès que possible  

Les cas urgents peuvent inclure :

- événement personnel imprévu  
- situation familiale urgente  

---

## 7. Refus ou modification de la demande

Une demande peut être refusée ou ajustée pour les raisons suivantes :

- contraintes opérationnelles  
- effectif insuffisant sur la période  
- période de forte activité  
- non-respect du délai de prévenance  

Le manager peut proposer :

- un report des dates  
- une modification partielle  

---

## 8. Confirmation de la demande

Une fois validée :

- l’employé reçoit une confirmation via le SIRH  
- les jours sont automatiquement déduits du solde  
- la période est enregistrée comme absence planifiée  

---

## 9. Modification ou annulation

Avant la date prévue :

- l’employé peut modifier ou annuler sa demande via le SIRH  
- une nouvelle validation du manager peut être requise  

---

## 10. Résumé opérationnel

- Les congés sont demandés via le SIRH  
- Le manager valide ou refuse la demande  
- Un délai de prévenance est recommandé  
- Les cas urgents doivent être signalés directement  
- Les demandes peuvent être modifiées ou refusées
