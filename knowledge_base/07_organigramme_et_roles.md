# Organigramme et Rôles Hiérarchiques

## 1. Objet

Ce document décrit la structure organisationnelle de l’entreprise, les rôles hiérarchiques et les modalités d’identification du responsable direct (N+1).

---

## 2. Structure de l’entreprise

L’entreprise est organisée en plusieurs directions et services :

- Direction Générale
- Ressources Humaines (RH)
- Direction Technique (IT)
- Direction Financière
- Direction Commerciale

Chaque direction est composée de plusieurs équipes, encadrées par des managers.

---

## 3. Notion de hiérarchie

### Manager (N+1)

Le manager, appelé aussi **N+1**, est le responsable direct du salarié.

Il est chargé de :

- superviser le travail quotidien
- valider les congés
- évaluer la performance
- accompagner le salarié

---

### Manager N+2

Le N+2 est le responsable du manager (niveau hiérarchique supérieur).

Il intervient généralement dans :

- décisions importantes
- arbitrages
- situations exceptionnelles

---

### Service RH

Le service RH intervient en support :

- gestion administrative
- validation de certaines demandes
- accompagnement des salariés et managers

---

## 4. Identification du responsable hiérarchique

Le salarié peut identifier son manager (N+1) via :

### Méthodes :

- le portail RH (SIRH)
- l’annuaire interne
- son contrat de travail ou fiche de poste
- communication interne (email, onboarding)

## 6. Cas particuliers

- En cas de changement de poste, le manager peut être modifié
- En cas de doute, le salarié doit vérifier via le SIRH ou contacter RH
- Certaines structures projet peuvent impliquer plusieurs responsables (fonctionnel vs hiérarchique)

---

## 7. Utilisation dans les démarches RH

Le manager (N+1) est impliqué dans :

- validation des congés
- validation des demandes spécifiques
- suivi des performances
- accompagnement professionnel

---

## 8. Résumé opérationnel

- Le N+1 est le responsable direct du salarié
- Il est identifiable via le SIRH ou l’annuaire interne
- Le manager valide les demandes courantes
- Le service RH intervient en support administratif
- La hiérarchie peut évoluer selon les changements de poste
